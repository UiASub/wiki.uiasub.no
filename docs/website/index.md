# UiASub Website Documentation

Welcome to the **UiASub** website documentation wiki! This comprehensive guide covers everything you need to know about developing, maintaining, and contributing to the UiASub website.

## About UiASub

UiASub is a student organization at the University of Agder focused on underwater robotics and ROV (Remotely Operated Vehicle) development. Our website serves as our digital presence and information hub.
